  MB_persistentAnimationKeys

Animation keys for animations that should be persisted. Inspect the animationKeys array to find valid keys for your layer.

@property (nonatomic, strong) NSArray *MB_persistentAnimationKeys

Discussion

Animation keys for animations that should be persisted. Inspect the animationKeys array to find valid keys for your layer.

CAAnimation instances associated with the provided keys will be copied and held onto, when the applications enters background mode and restored when exiting background mode.

Set to nilto disable persistance.
